# DRPS.NuK3.R

## Como usar:
  - Baixe o DRPS Nuker.exe
  - Clique aqui [aqui](https://discord.com/developers/applications)
  - Clique em "New Application" no canto superior direito
  - Customize como quiser
  - Vá na aba bot clique em reset token e copie o token
  - Vá em installation, na aba guild install adicione bot e administrator
  - Copie o link e adicione numa guild, após isso copie id da guild e de Enter.
